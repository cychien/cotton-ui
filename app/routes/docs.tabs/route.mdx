      ## Behaviors

      <b className='text-slate-400 text-sm'>Before hydration</b>
      - All tabs are implemented as \<a\> tags, so when a tab is clicked, the user will be redirected to the same page with different content

      <b className='text-slate-400 text-sm'>After hydration</b>
      - Supports keyboard navigation
      - Can access the active value from tabs when the active tab changes
      - Preserves all internal states, including user actions on tabs before hydration. For example, the checkbox state will be brought from the unhydrated world to the hydrated world
